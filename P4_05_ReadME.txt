Projet 4 - Analysez les ventes de votre entreprise

1 - Contenu

Dataset - contenant les 3 fichiers à traiter
Export - Contenant le dataframe après nettoyage
pictures - Contenant les graphiques qui seront utilisés pour la présentation

P4_01_scriptdonnées.ipynb - Notebook permettant le nettoyage des données et la création du dataframe
P4_02_scriptanalyse.ipynb - Notebook permettant l'analyse et la création de graphique pour la présentation
P4_03_OCR4.py - Fichier python pour l'execution des routines lors de l'analyse.
P4_04_Presentation.odp - Présentation des résultats
P4_04_Presentation.pdf - Présentation au format pdf pour la portabilité
P4_05_ReadME.txt - Ce fichier

2 - Utilisation

Les fichiers P4_01_scriptdonnées.ipynb et P4_02_scriptanalyse.ipynb nécessitent l'installation l'application Jupyter (https://jupyter.org)

P4_03_OCR4.py s'ouvre dans n'importe quelle application de traitement de texte. Il nécessite Python 3 pour se lancer, il est fourni par Jupyter

P4_04_Presentation.odp s'ouvre avec une application de présentation. Les graphiques y sont inserés comme lien, il est nécessaire d'avoir le dossier 'pictures' au même endroit que la présentation

P4_04_Presentation.pdf s'ouvre avec un viewer PDF ou un browser. Il est à utiliser en cas de problème de compatibilité avec le fichier P4_04_Presentation.odp

3 - Décompression

Il est préférable de garder la même structure de fichiers lors de la décompression du dossier zip afin d'éviter des problèmes de liens morts. En particulier, le dossier "pictures" qui contient les graphiques de la présentation, insérer en temps que lien dans la présentation
